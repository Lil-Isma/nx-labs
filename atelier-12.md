# Atelier 12 : Intégrer Storybook

Maintenant que nous avons des libraries partagées contenant nos composants réutilisables, il est pénible de devoir 
lancer toute l'application juste pour vérifier si nos modifications s'affichent correctement. Ce problème est 
particulièrement vrai pour les larges applications.

Nous allons intégrer Storybook pour développer et tester nos composants en isolation.

Suivre la [documentation officielle](https://nx.dev/docs/technologies/test-tools/storybook/introduction) pour
installer Storybook dans votre workspace Nx.
